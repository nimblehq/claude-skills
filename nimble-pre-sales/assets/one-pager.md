# One-Pager Template

The **One-Pager** is the standard internal document format used by the sales team to consolidate all relevant information about a project opportunity.

## Template

```markdown
# One Pager - {PROJECT_NAME}

Created On: {DATE}

# Scope

A very high-level project scope focused on the expected deliverables so that the team can clearly understand what we need to build or provide.

# Pain Point(s) & Business Goals

A brief explanation of why the lead wants to pursue this project.

# Constraints

List the known constraints (e.g., project timeline).

# Sales

## Context & Strategy

Explain how this deal needs to be approached and whether there are particular areas of focus. If there are some known compromises that would need to be made and are acceptable, detail it to provide all the necessary context to the team.

## Proposal Requirements

List the elements and resources that might be required for this proposal and which are not part of our base proposal template.
```
